# Техническое задание

**AI-платформа мониторинга недвижимости Krisha.kz для поиска инвестиционно привлекательных квартир**

| | |
|---|---|
| **Заказчик** | Жандос Кимадиев |

## Назначение системы

Автоматический поиск недооценённых квартир в Астане по выбранным жилым комплексам, накопление исторических данных, анализ рынка недвижимости, прогнозирование стоимости объектов и выдача инвестиционных рекомендаций через Telegram.

---

## 1. Общая информация

### 1.1 Цель проекта

Создать интеллектуальную систему анализа рынка недвижимости, способную:

- ежедневно отслеживать объявления;
- сохранять историю изменения цен;
- выявлять квартиры ниже рыночной стоимости;
- прогнозировать изменение цен;
- рассчитывать потенциальную прибыль;
- уведомлять пользователя через Telegram.

## 5. Хранение истории

Фиксировать:

- дату изменения цены;
- старую цену;
- новую цену;
- дату исчезновения объявления.

---

## 6. Анализ рынка

Для каждого ЖК рассчитывать:

- среднюю цену;
- медианную цену;
- среднюю цену за м²;
- медианную цену за м²;
- количество активных объектов;
- скорость продажи.

---

## 7. Поиск выгодных квартир

**Критерии:**

- цена ниже рынка на 10%;
- цена за м² ниже рынка на 10%;
- входит в ТОП-5 самых дешёвых объектов ЖК.
